Moved by Rawlssimple majority
The motion
To add a new rule
[221] A player may only award points to another player if the proposal is an amendment to a rule that provides for a point gain or is a new rule that provides for a point gain, and the point gain must be an increase in the maximum points that can be awarded or a decrease in the minimum points that can be awarded.

This ensures that point-awarding rules are used to refine the mechanics of the game rather than to create new, arbitrary point-farming opportunities.
