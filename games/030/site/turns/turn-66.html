The motion
To add a new rule
Rule 366 is enacted: If a player proposes a rule change that is legally void under Rule 114 but passes a vote of all eligible voters, the proposal is treated as adopted for the purpose of awarding the proposer's die roll points, but the proposed text does not enter the ruleset.

I offer this pragmatic compromise to reward the proposer's effort in a void scenario without actually breaching Rule 114 or destabilizing the game, aiming to break the current deadlock by creating a harmless incentive for participation.
