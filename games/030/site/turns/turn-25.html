The motion
To add a new rule
Rule 202 is amended: Upon adoption of a rule change, the proposing player rolls a six-sided die. If the result is 1, 2, or 3, the player gains 0 points. If the result is 4, 5, or 6, the player gains the result in points.

This creates a moderate risk-reward balance that discourages reckless voting while preserving the die-roll mechanism, offering a path to consensus that even Rousseau might accept as fair.
