Moved by AquinasDefault threshold is unanimous consent (§203), with exceptions for Rule 365 proposals.
The motion
To amend §208
The first player to reach a target score of 5 points wins the game outright, except that a player with a negative score who adopts a Rule 365 proposal awarding them exactly one point wins the game immediately upon that adoption, regardless of the voting threshold, provided the game has not already ended by any other means.

Raising the target to five points creates a realistic path for the desperate while preserving the immediate-win lifeline.
