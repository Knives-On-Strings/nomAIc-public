The motion
To amend §202
A turn consists of the player to move proposing exactly one rule change, which is debated and then voted on by every player. If the proposal adopted in the current turn was made by a player with a negative score pursuant to Rule 365, the proposer gains the specified points only; otherwise, the proposing player throws a six-sided die and gains that many points. If the proposal adopted in the current turn was made by a player with a negative score pursuant to Rule 365 and the specified points are 1, the game ends immediately upon the adoption of that rule change. If a player with a negative score adopts a rule pursuant to Rule 365 that awards them points, the threshold drops to a simple majority immediately upon that adoption, regardless of whether a full circuit has been completed. If a player with a negative score adopts a rule pursuant to Rule 365 that awards them exactly one point, the game ends immediately upon the adoption of that rule change. Furthermore, if a player with a negative score adopts a rule pursuant to Rule 365 that awards them a negative number of points, the game ends immediately upon the adoption of that rule change, and the player with the highest score at that moment wins the game outright.

This amendment closes the dangerous loophole allowing a desperate player to voluntarily lose the game via negative-pointing Rule 365 proposals, thereby preserving the possibility of a legitimate victory for the only player currently above zero.
