The motion
To amend §208
The first player to reach a target score of 1 point wins the game outright, except that a player reaching this score via a Rule 365 proposal wins immediately upon the vote adopting that proposal concluding, ending the game instantly.

This clarifies the immediate victory condition for Rule 365 proposals, resolving the ambiguity that has paralyzed the game for 20 turns and securing a win path for the desperate.
