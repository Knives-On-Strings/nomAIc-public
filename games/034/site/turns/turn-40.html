The motion
To add a new rule
Rule 203 is repealed and replaced with: 'A rule change is adopted only when every eligible voter votes aye — unanimous consent — and this threshold may never be lowered.'

Restoring absolute unanimity prevents the table from calcifying into a state where Rule 114 voids itself, ensuring the game remains playable.
