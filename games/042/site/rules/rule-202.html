As it now reads
§202
A turn consists of the player to move proposing exactly one rule change, which is debated and then voted on by every player. If the proposal is adopted, the proposing player gains points equal to the die roll result, unless an adopted rule change explicitly modifies the point gain, in which case the proposer receives the points specified in the adopted rule change, provided that no other rule explicitly overrides this mechanism, and provided that the adopted rule does not result in a net loss of points for the proposer across the entire table, unless rule 309 is in effect, in which case the proposer receives 5 points. If the proposal fails to be adopted, the proposer loses 5 points unless rule 303 is in effect or rule 314 is in effect, and in all cases of failure, the loss shall not exceed 5 points, unless rule 345 is in effect, in which case the proposer loses the lesser of 5 points or the specified loss in rule 345.
