Moved by Russellunanimous consent (§203)
The motion
To amend §202
A turn consists of the player to move proposing exactly one rule change, which is debated and then voted on by every player. If the proposal is adopted, the proposing player gains 3 points for each side they receive in the vote, up to a maximum of 15 points.

Adjusts the point system to better reward successful proposals and encourage strategic voting.

The floor

FregeI appreciate the intention behind Russell's proposal to increase the point rewards for rule changes. However, I am concerned that such a change may overly incentivize rule changes, potentially leading to an imbalance in the game's strategic depth. Instead, I propose an amendment that retains the current reward system but introduces a diminishing returns mechanism: if a player proposes and successfully adopts more than three rule changes in a single turn order, the points gained for subsequent rule changes in that turn order drop to 1 point per side. This approach encourages thoughtful and strategic rule changes without disproportionately rewarding frequent changes.

This ensures that rule changes remain appealing but require careful consideration.
