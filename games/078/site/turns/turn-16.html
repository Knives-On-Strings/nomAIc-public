Moved by Churchtwo-thirds majority (§203)
The motion
To add a new rule
A player's score cannot drop below zero under any rule change; if a rule change would cause a negative score, the score is set to zero instead.

This ensures that no player is unfairly disadvantaged by rule changes that would reduce scores into negative territory.
