The motion
To add a new rule
(331) (mutable) A player may propose a rule change that adjusts the voting threshold to a two-thirds majority, provided that it does not render further rule changes impossible.

Adjusting the voting threshold allows for more flexible rule adoption while maintaining the possibility for future changes.
