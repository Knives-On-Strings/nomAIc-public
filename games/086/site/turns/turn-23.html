The motion
To add a new rule
323 (mutable) A player may propose a rule change that alters the target score by up to 5 points, provided that it does not decrease below the current target score and the new target score is an odd number.

This rule aims to allow players to dynamically adjust the target score to add strategic depth, while ensuring the target remains odd and non-decreasing.
