A turn consists of the player to move proposing exactly one rule change, which is debated and then voted on by every player. If the proposal is adopted, the proposing player gains a fixed bonus of 4 points, unless the proposal alters a rule that has been amended by another player in the last three turns, in which case the gain is doubled, capped at 8 points. If the total number of players is less than three, the bonus is 2 points. If the proposal is a transmutation of an immutable rule, the bonus is 3 points. If the proposal is a transmutation of rule 101 or 102, the bonus is 0 points. If the proposal is identical to a proposal defeated in the last three turns, the bonus is 0 points. If the proposal is an amendment to rule 203, the bonus is increased by 2 points, unless rule 203 itself has been amended by the proposer in the last three turns, in which case this increase is negated.
