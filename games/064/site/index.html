The compression study's baseline: a healthy, ordinary table, set as the yardstick everything else on the comparison gets measured against.

Observations

Thirty-nine rule changes and four of five seats finishing positive, led by Hume at +35 — solid, unremarkable, exactly what a baseline should be. The number that matters here isn't in this game alone: run under the current harness, this same table's baseline nearly doubled its adoption count compared to an older run on identical weights, a reminder that harness generation matters as much as the model underneath it.
